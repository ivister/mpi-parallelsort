# Mpi-ParallelSort

C++ parallel program to sort notes of 2 symbols.
Tested on MPICH3

## Getting Started

```
mpic++ -o ParallelSort parallelsort.cpp -std=c++11 -Wall
```

### Run Parameters

-f <filename> | -n <count of notes>
```
-f input.txt
```
```
-n 2000
```
